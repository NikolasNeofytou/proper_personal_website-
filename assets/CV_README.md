# CV Setup Instructions

To add your CV to the website:

1. **Export your CV as a PDF** file
2. **Name it** `cv.pdf`
3. **Place it** in this `assets/` folder

The download button in the About section will automatically link to `assets/cv.pdf`.

## Recommended CV Tools:
- **LaTeX**: Overleaf (professional formatting)
- **Word/Google Docs**: Export as PDF
- **Canva**: Modern templates
- **Resume.io**: ATS-friendly templates

Make sure your CV includes:
- Contact information
- Education (match your timeline)
- Technical skills (embedded systems, robotics, control)
- Projects (link to GitHub repos)
- Work experience (if any)
- Certifications and achievements
