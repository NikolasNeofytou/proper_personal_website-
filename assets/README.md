# Assets Folder

This folder contains all the media assets for your portfolio website.

## Required Files

Please add the following files to make your portfolio complete:

### Images
1. **profile.jpg** - Your professional headshot
   - Recommended size: 400x400px
   - Format: JPG or PNG
   - Should be a professional photo with good lighting

2. **project1.jpg** - Screenshot of your first featured project
   - Recommended size: 600x400px or 1200x800px
   - Format: JPG or PNG
   - Show the main interface or key feature

3. **project2.jpg** - Screenshot of your second featured project
   - Same specifications as project1.jpg

4. **project3.jpg** - Screenshot of your third featured project
   - Same specifications as project1.jpg

### Documents
5. **resume.pdf** - Your professional resume/CV
   - Format: PDF
   - Keep it under 2MB for fast loading
   - Ensure it's up-to-date with your latest experience

### Icon
6. **favicon.ico** - Website favicon
   - Size: 16x16 or 32x32px
   - Format: ICO or PNG
   - Can be created from your logo or initials

## Tips for Great Images

### Profile Photo
- Use a professional headshot with a clean background
- Good lighting is essential
- Smile and look approachable
- Business casual or professional attire

### Project Screenshots
- Capture the most impressive features
- Use high-resolution images
- Consider showing the interface on different devices
- Add brief captions or annotations if helpful

## Image Optimization

Before adding images, consider optimizing them:
- Use tools like TinyPNG, ImageOptim, or Squoosh
- Convert to WebP format for better performance
- Ensure images are properly sized (don't use oversized images)

## Alternative: Using Placeholder Services

Until you have your own images, you can use:
- **Unsplash** (https://unsplash.com) - Free professional photos
- **UI Faces** (https://uifaces.co) - Profile picture placeholders
- **Lorem Picsum** (https://picsum.photos) - Random placeholder images

Example placeholder URLs (replace in HTML):
- Profile: `https://i.pravatar.cc/400`
- Projects: `https://picsum.photos/600/400?random=1`
